# juventus_xg_analysis
Analisi degli expected goals della Juventus prima e dopo l'arrivo di Spalletti, con visualizzazioni in Python e approccio sport analytics.

## Obiettivo del progetto

L'obiettivo di questo progetto è confrontare le prestazioni offensive e difensive della Juventus attraverso metriche avanzate come:

- xG (expected goals)
- xGA (expected goals against)
- xG Difference

L'analisi mette a confronto il periodo precedente e successivo all'arrivo di Spalletti, per evidenziare eventuali cambiamenti nel rendimento della squadra.

## Tecnologie utilizzate

- Python
- pandas
- matplotlib
- numpy
- soccerdata / Understat

## Dataset e fonte dati

I dati utilizzati provengono da Understat, recuperati tramite la libreria `soccerdata`.

## Grafici realizzati

Il progetto include visualizzazioni come:

- andamento degli xG partita per partita
- confronto tra xG creati e concessi
- media xG prima e dopo Spalletti
- differenziale xG nei due periodi
